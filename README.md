# Projeto Landing Page - The Odin Project

Este projeto foi desenvolvido como parte da minha jornada de aprendizado na plataforma **The Odin Project**. O foco principal foi replicar o layout proposto, aprimorando habilidades de estruturação e posicionamento.

## 🚀 Objetivo
- Praticar a estruturação semântica com **HTML**.
- Aplicar conceitos de **CSS** para replicar o design visual solicitado.
- Organização de elementos e uso de Flexbox para alinhamento.
